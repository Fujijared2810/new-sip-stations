Why We Made This

As a group of coffee lovers, we came up with the idea to allow fellow coffee enjoyers to easily find cafes near DLSU-D.

02

Target Audience

We made this site specifically for students, teachers, and staff for DLSU-D, but we hope any coffee lover near the university can benefit from it.

03

Programming Languages

This was started as a project for our programming class. We coded the site using HTML, CSS, and JavaScript.
